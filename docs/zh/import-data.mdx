## 从本地文件导入

从本地文件导入时，支持JSON/CSV/Excel三种格式的文件，可以通过拖拽或者点击按钮来上传。

对于JSON，格式为数组，数组中每个元素为一行，使用字典表示。样式如下：

```json
[
  {"field1": value, "field2": value, ...},
  {"field1": value, "field2": value, ...},
  ...
]
```

例如：

```json
[
  { "month": "January", "desktop": 186, "mobile": 80 },
  { "month": "February", "desktop": 305, "mobile": 200 },
  { "month": "March", "desktop": 237, "mobile": 120 },
  { "month": "April", "desktop": 73, "mobile": 190 },
  { "month": "May", "desktop": 209, "mobile": 130 },
  { "month": "June", "desktop": 214, "mobile": 140 }
]
```

CSV和Excel支持通用的格式。

## 输入文本导入

也可以直接在对话框输入文本来导入数据，点击「从文本」按钮，在弹出的对话框中，填入CSV或者JSON格式的数据。

数据导入后，显示为一个数据条。点击数据条可以查看数据内容，通过数据条后面的更新和删除按钮可以更新或者删除导入的数据。
